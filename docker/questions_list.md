# Docker Interview Questions

## Theory Questions
1. What is Docker and how does it differ from a hypervisor?
2. Explain the Docker architecture (Docker Engine, Daemon, Client, Registry).
3. What is the difference between a Docker image and a Docker container?
4. How do Docker volumes work and why are they used?
5. What is a Dockerfile and what are the most common instructions used in it?

## Practical Scenarios
6. **Scenario:** Your container is repeatedly exiting with code 137. How do you troubleshoot this?
7. **Scenario:** You need to optimize a Dockerfile to reduce the final image size. What techniques would you use?
8. **Scenario:** An application inside a container cannot connect to a database in another container. How do you debug the networking issue?
9. **Scenario:** You need to pass sensitive credentials to a container without hardcoding them in the image. How do you do this?
10. **Scenario:** A container's disk usage is growing rapidly, filling up the host. How do you find the cause and prevent it?
