# Metaphor Illustrator Style Guide
---
## Visual Language
- **Canvas:** 16:9, white background.
- **Lines:** Black, hand‑drawn style (stroke‑width ~2 px), mimicking the "小黑" aesthetic.
- **Accent colors:** Use a single highlight color (red, orange, or blue) for handwritten notes.
- **Text:** Minimal Chinese/English captions, hand‑written font style.
- **File naming:** `<slug>-<index>.svg` or `.excalidraw.json`.
