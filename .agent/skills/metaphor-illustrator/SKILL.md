## Usage
When invoked via Codex (or any AI agent that loads Codex skills), the skill expects:
1. The **text** of a `guide.md` (or the relevant theory question description).
2. Optional flags:
   - `--count N` – number of illustrations to generate (default 4).
   - `--format svg|excalidraw|png` – output format (default `svg`).

The skill will:
1. Extract the core concept from the input text.
2. Map the concept to a visual metaphor.
3. Generate a hand‑drawn style illustration using an image model (e.g., DALL·E or Stable Diffusion).
4. Post‑process with **Rough.js** to get a sketch look.
5. If `excalidraw` is requested, convert the SVG to Excalidraw JSON.
6. Save the resulting files under an `assets/` folder next to the corresponding `guide.md`.

## Example Prompt
```
Use metaphor-illustrator Generate 3 SVG illustrations for the following guide:
---
<paste guide.md content here>
---
```
