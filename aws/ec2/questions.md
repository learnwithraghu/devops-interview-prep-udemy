# Questions for EC2

## Theory Questions
1. What are the differences between instance types (e.g., t2, m5, c5) and how do you choose the right one?
2. Explain the lifecycle of an EC2 instance from launch to termination.
3. How does EC2 pricing work (On-Demand, Reserved, Spot, Savings Plans)?
4. What is an AMI and how would you create a custom AMI?
5. Describe the role of security groups vs. network ACLs for EC2 networking.

## Practical Scenarios
1. You need to troubleshoot an EC2 instance that is unreachable via SSH despite the security group allowing port 22. What steps do you take?
2. An instance is repeatedly being terminated due to Spot interruption. How would you handle this to maintain service continuity?
3. You must migrate a legacy application to a new EC2 instance with minimal downtime. Outline the migration process.
4. An instance is experiencing high CPU utilization. Explain how you would diagnose and mitigate the issue.
5. You need to create an AMI from a running instance and use it to launch a scaled‑out fleet. Describe the steps.
