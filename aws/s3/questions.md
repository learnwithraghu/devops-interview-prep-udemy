# Questions for S3

## Theory Questions
1. What are S3 storage classes and when would you use each?
2. Explain the concepts of versioning and lifecycle policies in S3.
3. How does S3 provide data durability and availability?
4. What is the difference between bucket policies and ACLs?
5. Describe how server‑side encryption works (SSE‑S3, SSE‑KMS, SSE‑C). 

## Practical Scenarios
1. A bucket is publicly accessible when it should be private. How would you investigate and remediate?
2. Objects are being overwritten unexpectedly. Which S3 features can prevent this?
3. You need to migrate a large dataset to a different region with minimal downtime. Outline the steps.
4. A client receives "AccessDenied" errors when uploading multipart objects. Diagnose the issue.
5. Implement a lifecycle rule to transition old logs to Glacier and delete after 365 days.
